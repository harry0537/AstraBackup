Abstract

Project Astra NZ presents an advanced autonomous agricultural rover system designed to revolutionize precision farming through intelligent navigation and real-time environmental sensing. This capstone project integrates RPLidar S3 360-degree scanning, Intel RealSense depth cameras, RTK GPS positioning, and Pixhawk flight control systems to create a robust platform for autonomous field operations. The system features comprehensive sensor fusion algorithms, obstacle avoidance capabilities, and a cloud-connected telemetry dashboard for remote monitoring and mission planning. Built on ArduPilot firmware with MAVLink communication protocols, the rover enables farmers to automate field surveys, crop monitoring, and precision agriculture tasks while maintaining reliable operation in dynamic outdoor environments.
